Проект написан на Ангуляре 11, primeng, highcharts
Описание:
Имеются родительские компоненты profession-rate-compliance, site-group-rate-compliance, site-rate-compliance
которые используют child element line-chart как шаблон для отрисовки линейного чарта.
В чарте есть reactive form состоящая из различных фильтров которые передаются через родительский компонент.
На каждое изменение в форме происходит http запрос к серверу для получения новых даных и перерисовки чарта.

Вопросы:
1) Нужен фидбек по структуре и качеству кода
2) Правильно ли реализована загрузка данных для фильтров через async await (так как эти данные должны прогрузиться первыми до отрисовки чарта) или можно как то сделать это через rxjs?
3) Также вопрос по проекту не относящийся к данному компоненту, правильно ли хранить authorization в cookie или local storage
